# CarGemini

CarGemini is a static web-based project that provides information about various luxury and sports cars. The project is designed with a clean and simple user interface using HTML and multimedia elements. It was developed as part of a Semester-III academic project.

---

## Features

- Home / Intro page with attractive visuals  
- Car details page showcasing different luxury cars  
- Car videos page for better visual experience  
- Contact page  
- High-quality images of cars  
- Simple and responsive layout  

---

## Technologies Used

- **HTML5**
- **CSS3** (basic styling)
- **Images & Multimedia (WebP, JPG, PNG, AVIF)**
- **Static Web Design**
